## Parcelable
* 子类继承需要重写 

  ```
  Parcelable.CREATOR
  ```

   接口;
* 如果子类没有新增字段则不需要重写构造方法;
* 有新增字段需要重写构造方法和,

  ```
  Parcelable.writeToParcel()
  ```

  加上新增的字段
